# Introduction
## 🎨 VibePrinting - Art Studio

VibePrinting 是一款轻量级的、基于浏览器的视觉实验工具。它将现代 **AI 图像分割技术** 与复古的 **丝网印刷（Screentone/Halftone）美学** 相结合，让用户一键创作出具有 Y2K 复古美感的艺术作品。

## ✨ 核心亮点

- **🤖 AI 驱动的智能抠图**：集成 `TensorFlow.js` (BodyPix 模型)，在浏览器端实现零延迟的人像/物体提取，无需上传服务器，保护隐私。
- **🏁 动态丝网滤镜**：自定义开发的 Halftone 算法，支持实时调整网点密度（Dot Size）与对比度，模拟真实的丝网印刷质感。
- **🦋 Y2K 贴纸系统**：内置全屏覆盖式贴纸模版（星星、复古 Y2K），一键增强视觉冲击力。
- **🎨 莫兰迪色系 UI**：精心设计的极简莫兰迪配色方案，提供沉浸式的创作体验。
- **🎲 随机灵感生成**：一键随机化所有参数与配色，探索意想不到的视觉组合。
